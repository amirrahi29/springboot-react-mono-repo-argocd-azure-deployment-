## ⚙️ Tech Stack

- **Backend:** Spring Boot (Java)
- **Frontend:** React.js
- **DevOps:**
  - Docker
  - Kubernetes (AKS)
  - ArgoCD (GitOps)
- **Cloud:** Microsoft Azure

---

## 📦 Deployment Flow

1. Developer pushes code to GitHub
2. Docker images are built
3. Kubernetes manifests are updated
4. ArgoCD detects changes from Git repo
5. Automatic deployment to AKS 🚀
